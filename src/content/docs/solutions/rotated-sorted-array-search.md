</style><svg style="display: none;"><defs id="MathJax_SVG_glyphs"></defs></svg><p>You are given an integer array 'nums' sorted in ascending order with distinct values. The array may have been rotated at an unknown pivot index 'k' such that the resulting array becomes:</p>

<p>Before rotaion :
[ nums[0],nums[1],...,nums[k−1],nums[k],nums[k+1],...,nums[n−1] ]</p>

<p>After rotaion :
[ nums[k],nums[k+1],...,nums[n−1],nums[0],nums[1],...,nums[k−1] ]</p>

<p><strong>Example</strong> <br>
nums = [5,6,7,8,9,10,11,12] might be rotated at pivot index 3 and become , nums = [8,9,10,11,12,5,6,7]</p>

<p><strong>Task</strong></p>

<p>Given the rotated array 'nums' and an integer 'target', your task is to find the index of 'target' in 'nums'. If 'target' is found, return the index. If 'target' is not found, return -1.</p>
